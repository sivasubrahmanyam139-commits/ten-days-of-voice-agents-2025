# NEXUS - Personal AI Voice Assistant

"NEXUS online. Ready to assist, sir."

NEXUS is a Jarvis-inspired personal AI voice assistant built for the Murf AI Voice Hackathon 2026. Just speak naturally and NEXUS responds instantly with a lifelike human voice.

## What it does
- Listens to your voice in real-time
- Understands natural language
- Responds with a natural human voice powered by Murf Falcon TTS
- Holds full conversations like a real AI companion

## Tech Stack
- Murf Falcon TTS — ultra-fast lifelike voice
- LiveKit Agents — real-time audio streaming
- Deepgram STT — accurate speech recognition
- Google Gemini — intelligent AI reasoning
- Python — backend agent

## How to run
1. Clone this repo
2. Add your API keys to backend/.env.local
3. Run: py -3.11 -m uv run python src/agent.py dev
4. Open agents-playground.livekit.io and connect!
